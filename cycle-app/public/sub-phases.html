Sub-Phase Awareness

How treatment-specific phases change the emotional tone of daily content

Current — Generic Phases (All Treatments)
Right now, we divide any cycle into 5 generic phases based on progress percentage. This works but misses the emotional reality of specific treatment milestones.

Day 1
Fresh start, hopeful
EARLY

Grounding, patient
LATE
~55-75%
Calming, reassuring
FINAL
~75%+
Celebratory, tender
Proposed — IVF (18 days)
IVF has distinct medical milestones that create very different emotional experiences. A woman on day 2 (just started injections) feels completely different to day 10 (egg retrieval) vs day 15 (2-week wait).
STIMULATION
Days 1-9
Daily injections. Body changing. Brave, uncomfortable, determined.
TRIGGER & RETRIEVAL
Days 10-11
The big moment. Nervous, hopeful, physically sore.
EMBRYO WAIT
Days 12-13
Waiting for lab results. Anxious, out of control, need distraction.
TRANSFER
Day 14
Transfer day. Still, hopeful, sacred.
TWO-WEEK WAIT
Days 15-18
The hardest part. Every symptom means something. Gentle, distracting, validating.
Day 3 — Stimulation phase
🔥 TODAY'S FIRE
"Your body is doing extraordinary things right now, even when it doesn't feel like it."
— affirmation would reference injections, bloating, physical courage
🍳 YOUR FUEL
High-protein + hydration focus (reduces bloating during stims)
Day 15 — Two-week wait
🔥 TODAY'S FIRE
"You don't need to know the answer today. You just need to breathe through this moment."
— affirmation would acknowledge the wait, uncertainty, need for patience
🍳 YOUR FUEL
Comfort foods + anti-inflammatory focus (pineapple core, warm soups)
Proposed — Egg Freezing (14 days)
PREP
Days 1-2
Starting meds. Empowered choice, maybe nervous.
STIMULATION
Days 3-10
Daily injections. Bloating, fatigue. Focus on why she's doing this.
MONITORING
Days 11-12
Frequent scans. Almost there. Anticipation.
RETRIEVAL & RECOVERY
Days 13-14
It's done. Rest, pride, relief. Body needs care.
Proposed — Frozen Embryo Transfer (18 days)
LINING PREP
Days 1-10
Taking estrogen, patches. Quieter than IVF stims. Waiting, preparing.
PROGESTERONE
Days 11-13
Started progesterone. Body prep intensifies. Transfer approaching.
TRANSFER
Day 14
The moment. Sacred, still, hopeful.
TWO-WEEK WAIT
Days 15-18
Same as IVF — the hardest emotional stretch.
How to Implement
Replace the generic getCyclePhase(dayNumber, totalDays) function with treatment-aware sub-phases:

getCyclePhase(dayNumber, totalDays, treatment)

Each treatment type gets its own phase map with specific day ranges and emotional tones. These tones are passed to the AI prompt, which already accepts phase + tone context. No new UI needed — just richer, more specific AI-generated content.
// Example: IVF sub-phases
if (treatment === 'ivf') {
  if (day <= totalDays * 0.5) return { phase: 'stimulation', tone: 'brave, physical courage...' }
  if (day <= totalDays * 0.6) return { phase: 'retrieval', tone: 'nervous, hopeful...' }
  if (day <= totalDays * 0.75) return { phase: 'embryo-wait', tone: 'anxious, distraction...' }
  return { phase: 'two-week-wait', tone: 'gentle, validating...' }
}
